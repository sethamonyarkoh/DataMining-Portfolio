# Titanic Exploratory Data Analysis

## Project Overview
This project explores the Titanic dataset to understand the factors that influenced passenger survival using exploratory data analysis techniques.

## Dataset
The dataset comes from the Kaggle Titanic competition.

## Project Structure
- Data/ – datasets (raw and cleaned)
- Notebooks/ – Jupyter notebooks for analysis
- Scripts/ – Python scripts
- Models/ – trained models
- Reports/ – final reports and figures
- References/ – documentation and sources

## Objectives
- Clean and prepare the data
- Explore relationships between features and survival
- Visualize important patterns
- Summarize key insights

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook
